# Changelog

## v1.0.0 - 2026-03-19

首个可协作试跑版本。

### 新增

- 投放录入、销售承接、学员管理、风险预警、退款工作台、ROI 分析全链路模块
- 账号管理与账号设置
- 退款审批链路、同意节点与处理留痕
- 营收台账 `RevenueLedger`
- 审计日志 `AuditLog`
- 健康检查接口 `/api/health`
- 局域网启动脚本与 Docker 部署文件

### 优化

- 学员主档不再允许直接推进退款流程状态
- 同手机号允许多条线索实例
- 同手机号学员可新增多期报名记录
- 风险页加入自动预警信号
- ROI 支持营期、截止日期、预收净收三种口径

### 部署

- 支持局域网单机试跑
- 支持 PostgreSQL + Docker Compose 云端部署
